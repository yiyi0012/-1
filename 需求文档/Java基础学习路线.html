从环境入门到核心语法，循序渐进掌握 Java 基础。

Java 学习路线

Java 基础学习路线 一、环境入门（1 - 2 周） 1. 搭环境：装 JDK 11/17，配 JAVA_HOME 和 Path，用 java -version 验证。 2. 选工具：先手写代码（Notepad++），再学 IDEA 基础操作（建项目、运行）。 3. 写程序：编 Hello World，搞懂类结构、main 方法（程序入口）、System.out.println()（输出）。 二、核心语法（4 - 6 周） 1. 基础语法（3 - 4 天）  变量：掌握基本类型（int/double/boolean），会定义赋值（int a=10）。  运算符：用算术（+/*/%）、逻辑（&&/||）、比较（>/==）写简单计算。  注释：会单行（//）、多行（/* */）注释。 2. 流程控制（5 - 7 天）  分支：用 if - else（多条件判断）、switch（固定值匹配）处理逻辑。  循环：用 for（已知次数，如遍历 1 - 100）、while/do - while（未知次数，如键盘输入）。 3. 数组（3 - 4 天）  一维数组：会定义（int[] arr={1,2,3}）、遍历（普通 / 增强 for 循环）。  二维数组：理解嵌套结构（int[ ][ ] arr={{1,2},{3,4}}），会双层遍历。  操作：练求最值、冒泡排序、元素查找。 4. 面向对象基础（1 周）  类与对象：懂 “类是模板，对象是实例”，会定义成员变量（name）、成员方法（study()），调用对象方法。  构造方法：会写无参 / 有参构造，用 this 区分成员与局部变量。
